# Suman - Quran Memorization App

A web application designed to help users memorize the Quran through interactive learning techniques, progress tracking, and spaced repetition.

## Features (Planned)
- 📖 Display Quranic verses (Surahs)
- 🎯 Spaced repetition for effective memorization
- ✅ Track memorization progress
- 🔊 Audio support for Quranic recitation
- 🌙 Dark mode support
- 📊 Statistics and progress dashboard
- 🎨 Beautiful, user-friendly interface

## Tech Stack
- **Frontend:** React + Vite
- **Styling:** Tailwind CSS
- **Data:** Quran API (public data)
- **Storage:** Local Storage / Firebase (future)

## Getting Started

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/suman19ise-ui/Suman.git
cd Suman
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open `http://localhost:5173` in your browser
